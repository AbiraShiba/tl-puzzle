# tl-puzzle

タイムラインにスキルを配置して、効果（バフ/デバフ/攻撃）と対象を可視化・編集するツールです。

## できること
- 生徒/敵のステータス編集
- EXをドラッグ&ドロップでタイムラインに配置
- 効果ごとに対象（自分/指定生徒・敵/全員/敵のみ）を個別指定
- 効果の種類（バフ/デバフ/攻撃）と数値/継続時間を編集
- 効果の重なりをタイムライン/一覧で確認
- URL共有（状態とデータをエンコード）

## はじめかた
1. 依存関係を入れる  
   ```bash
   npm install
   ```
2. 開発サーバーを起動  
   ```bash
   npm run dev
   ```
   表示されたURL（例: http://localhost:5173）をブラウザで開きます。
3. ビルド  
   ```bash
   npm run build
   ```
   `dist/` が出力されます。

## 操作メモ
- EXパレットのボタンをドラッグしてタイムラインに配置します。
- EXブロックはドラッグで移動、右端ハンドルで継続時間を調整できます。
- ブロックを選択すると、対象/開始秒数/継続時間を編集できます。
- 効果バーをクリックすると、その効果だけの対象を設定できます。
- 敵は対象として選択できます（指定生徒/敵の一覧に含まれます）。
- 「時間解像度」「戦闘時間」を設定できます。
- 効果の重複は `stackGroup` が同じ場合のみ後書き、それ以外は加算です。
